# Example MATLAB Concepts Used

## Matrix Multiplication

```matlab
A = [1 2 3; 4 5 6; 7 8 9];
B = [10 11 12; 13 14 15; 16 17 18];

C = A * B;
```

## For Loop

```matlab
for i = 1:10
    disp(i);
end
```

## Conditional Statement

```matlab
if (x > y)
    disp('x is greater');
else
    disp('y is greater');
end
```
